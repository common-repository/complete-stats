This plugin creates a sidebar widget with your website statistics in it - alexa, pr, etc.

== Description ==

This plugin, when activated, creates it's own sidebar widget with a bunch of stats about your website in it, currently including PR and Alexa rank, with more being added.


== Installation ==

It is very simple to get the plugin working on your blog.

1. Upload `complete-stats.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Put the widget where you want it in the widgets manager.
